# **App Name**: ProfolioCraft

## Core Features:

- Dynamic Content Display: Presents portfolio projects in a visually appealing manner, utilizing the defined design system for consistent styling.
- Interactive Navigation: Enables seamless browsing through portfolio sections with smooth transitions and functional links.
- Contact Form: Implements a functional contact form for visitors to send messages, adhering to the specified input styles and layout.
- Animated Transitions: Applies subtle animations, like fade-in effects and hover interactions, to enhance user experience and visual appeal.

## Style Guidelines:

- Primary gradient: Blends '#FF6B6B' to '#FFD93D' to create a vibrant and energetic feel.
- Accent gradient: Transitions from '#6BCB77' to '#4D96FF' for highlights and interactive elements.
- Background: White ('#FFFFFF') provides a clean and modern base.
- Font: 'Poppins', sans-serif provides a balanced and modern appearance.
- The design implements a responsive design.
- Subtle button hover animation using scale(1.05).
- Card hover effect to elevate cards using translateY(-4px).
